To compile and run program Java programs, the Java Development Kit (JDK)
must be installed on your machine (JDK 23+, version compatibility may vary).
Use a command-line interface and ensure the active directory contains 
all the .java files contained in Stage 3.

To compile Main.java use command:
javac Main.java


After compiling, use command:
java Main.java

to run the program. See userguide.pdf for information on using the program.
